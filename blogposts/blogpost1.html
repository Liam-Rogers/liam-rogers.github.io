Boolean-State driven buttons in SwiftUI

In this article, I will cover how you can make buttons have action in SwiftUI. First of all, we will create a button that when pressed will open a sheet view containing either some defined elements, or you could embed another view inside.

            
import SwiftUI

struct ContentView: View {
    @State var buttonAction = false
    var body: some View {
        Button("This is a button") {
            buttonAction.toggle()
        }
        .padding()
        .background(Color.red)
        .sheet(isPresented: $buttonAction, content: {
            Text("This is a sheet view")
        })
    }
}
